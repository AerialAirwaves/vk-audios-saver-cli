VK Audio Sync for *nix

Software dependencies:
Python 2.7 with loadable modules: os, time, urllib2, json, codecs, re, HTMLParser, mutagen (!), urlgrabber (!)
Installed *NIX programs: bash, find, sort, sed

P.S. This program stable working on my home server, that running Ubuntu 14.04.

Warning! Before executing sync.py edit config.py and set your account login information and download destination dir.
